**Примечания к релизу:**

Обязательно обновите центральную консоль, если используете Ideco NGFW версии 17.5. Эта версия Ideco Center совместима только с этой версией Ideco NGFW.

- Создание и синхронизация с серверами Ideco NGFW версии 17.5 профилей Контроля приложений;
- Использование провилей Контроля приложений в правилах Файрвола.
