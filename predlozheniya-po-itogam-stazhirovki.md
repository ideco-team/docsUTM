# Предложения по итогам стажировки

## Предложения по итогам стажировки

Стажировка - это обучение на рабочем месте, приобретение практического опыта под руководством наставника.

{% hint style="info" %}
Стажировка в Айдеко включает в себя знакомство с коллегами, настройку необходимых для работы приложений и сервисов, ознакомление с рабочими процессами, а также практику по работе с продуктами, внутренними сервисами и текстами.
{% endhint %}

В ходе прохождения двухнедельной стажировки в Айдеко определены следующие характеристики документации для нового сотрудника, затрудняющие прогружение в рабочие процессы:

1\. Недостаток терминов в тексте документации. Проблема - затраты времени на поиск информации в поисковиках.

2\. Краткое описание задач. Проблема - затраты времени на самостоятельный поиск решений и уточнение ожидаемого результата.

3\. Недостаток контекста. Проблема - риск неверного понимания задачи и особенностей продукта, использование недостоверных источников.

Для оптимизации процесса погружения в рабочие процессы предлагаются следующие решения:

1. Включить определения узкоспециализированных терминов в тексты статей, например, с помощью цветных сносок, и/или сделать гиперссылки на Глоссарий и дополнить его.
2. Расширить описание задач, подробнее раскрыть алгоритм действий и ожидаемый результат.
3. Включить описание продуктов и проблем, которые решает пользователь с их помощью, описать возможности применения функций, добавить перечень полезных для выполнения задачи источников.

Следует отметить положительные характеристики документации для нового сотрудника. Во-первых, последовательное расположение текстовых блоков для постепенного освоения информации: от теории к практике, от простого к сложному и т.д.\
Во-вторых, отсутствие информационного шума, краткое изложение сути материала. В-третьих, к блоку для нового сотрудника прилагаются ссылки на дополнительные материалы (курсы, конференции, полезные внешние инструменты). Вся информация, предложенная для изучения в первые две недели работы, оказалась практикоориентированной и полезной, однако потребовалось воспользоваться поисковиками и помощью коллег.
