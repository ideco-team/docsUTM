# Netflow

![](/.gitbook/assets/netflow.png)

* **Версия протокола** - выберите версию протокола Netflow. Доступны Netflow 5, Netflow 9 (по умолчанию) и Netflow 10 (IPFIX);
* **Интерфейсы учета трафика** - выберите интерфейсы, трафик которых хотите мониторить (доступны Ethernet-интерфейсы, Ethernet + PPTP/L2TP/PPPoE, GRE, локальный VPN-трафик, IPsec, GRE over IPsec). Если не заполнены, статистика не будет экспортироваться. Изменить индекс Netflow можно в настройках сетевых интерфейсов, значение по умолчанию - 0;
* **Интервал отправки для активного потока, секунды** - введите ... , по умолчанию - 300 секунд;
* **IP-адрес коллектора** - введите IP-адрес коллектора данных. Если не заполнен, статистика не будет экспортироваться;
* **Порт коллектора** - введите номер UDP-порта коллектора данных, используемого для приема пакетов Netflow;
* **Интервал отправки шаблона, пакеты** - по умолчанию - 20;
* **Интервал отправки шаблона, секунды** - по умолчанию - 1800.

ВАЖНО: на интерфейсах WAN + PPTP/L2TP/PPPoE будет учитываться трафик на PPP интерфейсе (Eppp), а не на родительском Veth
